Periodic Table using HTML and Bulma CSS

Why the Periodic Table?

The Periodic Table is one of the most significant achievements in Science. Dmitri Ivanovich Mandeleev started working on the Periodic Table in 1869. Since then, many people have contributed to it in various forms. It has been a staggering 150 years after the Periodic Table has come into existance. To honour it, I decided to make a Periodic Table using HTML and Bulma CSS that would give you everything you need to know about the Periodic Table.
You can check it out here

How does it help?

Since I'm always working with chemicals such as Hydrochloric acid and other stuff like that, it always used to be frustrating for me to remember the Atomic Masses, Weights, Electrons, Protons, Neutrons and other things related to the Periodic Table. But after I started using this, it has become so easy and now after regular usage I remember almost all the elements in the Periodic Table.

An easy way to remember

I recently found that remembering anything is easier by memorising it like a song or a story which you can relate to than mugging it up. So, I recently came across this song which I would like to share with you that is easy to sing and covers all the Elements of the Periodic Table.
